Methods

(async, inner) getProducts(page, category, search, orderByPrice, page) → {Promise}

Esta funcion recibe los parametros solicitados por el usuario y devuelve los productos que coinciden con los parametros recibidos.
Parameters:
Name Type Description
page Number Número que representa la página que el usuario desea visualizar.
category String Categoría de productos que desea visualizar el usuario.
search String Parámetro de búsqueda del nombre del producto ingresado por el usuario.
orderByPrice String Parámetro que ordena los productos por precio de mayor a menor y viceversa.
page Number Descuento que sirve para filtrar la búsqueda de productos con descuentos del 10% o 20%.
